Local Anonymizer

Privacy · Data minimization · Local-first
Open source · MIT

Strip personal data from documents before they reach any cloud LLM – then restore the result locally and deterministically. You drop a document in, sensitive data is replaced by stable placeholders (⟦NAME_1⟧, ⟦IBAN_1⟧), and you get a clean file you can safely paste into ChatGPT, Claude or any cloud tool. The mapping back to the real values never leaves your machine.

  • Hybrid detection: deterministic regex (email, IBAN, card, passport, address, phone) plus an optional local LLM for free-text names.
  • Deterministic restore via exact DB lookup – reproducible and auditable, never an LLM guess.
  • Formats: .txt, .md, .pdf, .docx and scans via OCR. Tuned for European / DACH formats.
  • Zero core dependencies, runs fully offline, with a one-click macOS launcher.
Python SQLite Regex + local LLM OCR (Tesseract) CLI + macOS launcher
